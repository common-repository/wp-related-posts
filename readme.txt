== Installation ==

1. Upload `wp-relatedposts.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change settings in admin panel

== Frequently Asked Questions ==

Q: How related on category? 
R: Enter in options,menu WP Related Posts, take Related on: Category.

Q: How related on tags? 
R: Enter in options,menu WP Related Posts, take Related on: Tags.
